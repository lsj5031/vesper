# Vesper

"Where the day settles."

An elegant, sophisticated, completely offline-first RSS reader that looks and feels exactly like reading the Financial Times in 2025.

## Quick Start

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Features
- Offline-first (IndexedDB + PWA)
- Three-panel FT.com inspired layout
- OPML Import/Export
- FT Origami O3 design tokens with custom fonts
- Salmon accents & perfect typography

## Tech Stack
- SvelteKit 2
- Dexie.js
- Tailwind CSS + FT Origami O3
- Custom fonts: Playfair Display & IBM Plex Sans
